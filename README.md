# JDS
Java Desktop System - Trying to revive created by Sun Microsystems and forgotten by Oracle - Desktop in Java.

Just a few apps available now, needs jdk installed and environment variable setup.

Run: # java -jar RetroOS.jar  
Or: # java Main

To get original Java Look & Feel if not shown by default: 
Try # java -Dswing.defaultlaf=javax.swi.plaf.metal.MetalLookAndFeel Main
Have fun ...
